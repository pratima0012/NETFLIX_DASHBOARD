🎬 Netflix Data Analysis Dashboard (Power BI)
📌 Overview

This project is an interactive Netflix Dashboard built using Power BI to analyze content trends, user ratings, and audience engagement.

🎯 Objectives
Analyze Netflix content growth over the years
Understand distribution of content by rating
Explore relationship between user ratings and popularity
Create an interactive and visually appealing dashboard
📊 Key Features
📈 Content Growth Over Time (Line Chart)
🍩 Content Distribution by Rating (Donut Chart)
📊 Average User Rating by Rating Level (Bar Chart)
📋 Top Content Analysis (Table)
🎯 KPI Cards:
Total Titles
Average User Rating
Total User Ratings
🎛️ Slicer for filtering by Release Year
🧠 Key Insights
Mature content tends to receive higher user ratings
Netflix content has significantly increased in recent years
Higher engagement often aligns with better ratings
🛠️ Tools Used
Power BI
Power Query
DAX
Data Visualization
📂 Dataset

The dataset contains:

Title
Content Rating
Release Year
User Rating Score
Number of User Ratings
🚀 How to Use
Download the .pbix file
Open it in Power BI Desktop
Use slicers and visuals to explore insights
